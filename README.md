# Student Record Management System

## Description

The Student Record Management System is a Java application developed using Object-Oriented Programming (OOP) principles and Java File I/O. The system allows users to manage student records by adding, searching, updating, deleting, and displaying student information. It also generates reports based on student GPA data and demonstrates the use of text files, binary files, object serialization, buffered streams, and exception handling.

---

## Features

- Add Student
- Search Student by ID
- Update Student Information
- Delete Student
- Display All Students
- Generate Student Reports
- Create Backup Files
- Display File Properties

---

## Student Information

Each student record contains:

- Student ID
- Name
- Department
- GPA

## Project Structure

---

## Class Responsibilities

### Student.java
Represents a student object and stores student information.

### FileHandler.java
Handles file operations including:
- Text file handling
- Binary file handling
- Object serialization
- Backup creation
- File information display

### StudentManager.java
Manages student operations such as:
- Add Student
- Search Student
- Update Student
- Delete Student
- Display Students
- Generate Reports

### Main.java
Provides a menu-driven user interface and allows users to interact with the system.

## Report Generation

The system generates reports containing:

## Exception Handling

The program handles common errors such as:

- Invalid user input
